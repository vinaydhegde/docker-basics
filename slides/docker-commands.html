DOCKER COMMANDS


Helpful commands:

  1. docker cp: Copy files/folders between a container and the local filesystem
  2. docker commit: Create a new image from a container's changes
  3. docker diff: Inspect changes to files or directories on a container's filesystem
  4. docker export: Export a container's filesystem as a tar archive
  5. docker history: Show the history of an image
  6. docker info: Display system-wide information
  7. docker inspect: Return low-level information on Docker objects
  8. docker login: Log in to a Docker registry
  9. docker kill: Kill one or more running containers
  10. docker system prune: Remove unused data (it can be unused images/stopped containers/volumes/networks)
  11. docker top: Display the running processes of a container
  12. docker logs: Fetch the logs of a container
  13. docker update: Update configuration of one or more containers (For example CPU, Memory etc.)
  14. docker stats: Display a live stream of container(s) resource usage statistics (CPU usage & Memory usage)
  15. docker system df: Show docker disk usage (of images, containers & local volumes)
