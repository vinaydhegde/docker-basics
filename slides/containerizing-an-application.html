CONTAINERIZING AN APPLICATION

Create and test individual containers for each component of your application by creating Docker images

  1. Clone an example project from GitHub (This is a simple bulletin board application, written in node.js)
    git clone -b v1 https://github.com/docker-training/node-bulletin-board
    cd node-bulletin-board/bulletin-board-app
  2. Build & Test your Image
    docker image build -t bulletinboard:1.0
  3. Start a container based on your new image
    docker container run --publish 8000:8080 --detach --name bb bulletinboard:1.0
  4. Visit your application in a browser at localhost:8000
